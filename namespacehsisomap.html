◆ KNNGraphWithImplementation()

std::shared_ptr<KNNGraph> hsisomap::KNNGraphWithImplementation ( KNNGraphImplementation  knngraph_implementation,
std::shared_ptr< gsl::Matrix >  data,
PropertyList  property_list 
)

Return the implementation class and construct the kNN graph with the specified implementation type.

Note that the graph construction usually happens in the constructor. Therefore this function also essentially performs the graph construction.

Parameters
knngraph_implementationthe kNN graph construction method with one of KNNGraphImplementation.
datathe data matrix (the pairwise distance matrix).
property_listspecify the property list used by a certain method of kNN graph. The key and value pairs could be different for different method.
Returns
smart pointer to the implementation class of the kNN graph with the specified implementation type.
