Detailed Description

A simple wrapper class for the OpenCL implementation of the Parallel Dijkstra Algorithm. for all-pair or selected-pair shortest distances calculation.

This code is adapted from the example code (by Dan Ginsburg) for the book OpenCL Programming Guide (by Munish, et al.) (http://www.heterogeneouscompute.org/?page_id=5) with some tweaks, one data reading synchronization bug fixed for large datasets, and multiple source shortest distances support added. Both of this code and the original code implemented the parallel Dijkstra Algorithm from the paper "Accelerating large graph algorithms on the GPU using CUDA" (by Harish and Narayanan).
